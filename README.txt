Low-fidelity Python Wumpus Logic Agent (Flask)

Run:
  1) python -m pip install -r requirements.txt
  2) python app.py
  3) Open http://127.0.0.1:5000

Notes:
- Single-user in-memory game state (sufficient for assignment demo).
- Backend does: random hazards, percept generation, KB (CNF), resolution refutation.
- Frontend is intentionally low-fidelity.
